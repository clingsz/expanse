## 🎮 最新设计变更 (2025-12-07)

### 关键决策
- **军事系统简化**: 移除兵营、军工厂、控制塔等专用军事建筑
  - 所有子弹和无人机在**组装机**中生产
  - 基础军事科技仅解锁配方，不解锁建筑
  - 基础无人机容量：100 (无需控制塔)

- **建筑槽位**: 统一为 **16 槽位/区域** (原10槽位过少)

- **科研系统 (DEBUG模式)**:
  - 研究中心速度：10x (基础: 10.0, 高级: 20.0)
  - 所有科技时间：除以10
  - 消耗逻辑：Factorio风格连续分数消耗

- **初始资源 (DEBUG)**:
  - 铁板: 500, 铜板: 300, 石头: 200
  - 齿轮: 100, 电路: 50, 钢板: 100
  - 煤炭: 100

- **界面优化策略**:
  - 完整重建：仅在打开界面或解锁科技时
  - 轻量级更新：游戏循环中仅更新数值
  - 避免DOM闪烁：使用data属性精确定位

## 一、游戏概述

### 核心概念
《扩张前线》是一款极简版的 Factorio 风格游戏，去除了地图和物流系统，专注于**生产链优化**、**科技研发**、**资源管理**和**区域推进**。玩家通过建造自动化工厂，生产无人机和弹药，逐个区域征服虫族占领的星球。

### 核心循环
```
采集资源 → 建造工厂 → 生产物资 → 研究科技 → 
制造无人机 → 占领新区域 → 获得更多槽位和资源 → 循环加速
```

### 游戏目标
- **主线目标：** 征服 20 个区域，击败 4 个 BOSS，通关游戏
- **优化目标：** 建立高效生产链，快速推进
- **可选目标：** 无尽模式，挑战极限区域

## 二、核心系统

### 2.1 区域系统

#### 区域结构
- **总计 20 个区域** = 4 个阶段 × 5 个区域
- 每 5 个区域一个 BOSS 关卡
- 必须线性推进，无法跳过

#### 区域属性
```
区域编号：1-20
槽位数量：10-20（逐渐增加）
资源点：铁矿、铜矿、煤矿、原油、铀矿等
敌人配置：虫族类型和数量
```

#### 四个阶段

**阶段 1：基础工业时代（区域 1-5）**
- 资源：铁、铜、煤
- 科技：自动化、基础武器
- BOSS：装甲巨虫（需要穿甲弹）
- 奖励：原油区域 + 化工科技解锁

**阶段 2：化工时代（区域 6-10）**
- 新资源：原油（随机出现）
- 新产物：塑料、电池、炸药
- 科技：激光武器、高级组装
- BOSS：能量核心虫（需要激光武器）
- 奖励：铀矿区域 + 核能科技解锁

**阶段 3：核能时代（区域 11-15）**
- 新资源：铀矿（随机出现）
- 新产物：核燃料棒、高级材料
- 科技：核武器、高级自动化
- BOSS：虫巢母体（持久战）
- 奖励：稀有矿区域 + 终极科技解锁

**阶段 4：终极时代（区域 16-20）**
- 新资源：稀有元素
- 新产物：反物质、纳米材料
- 科技：终极武器、聚变能源
- BOSS：虫族主宰（三阶段战斗）
- 奖励：通关 + 无尽模式解锁

### 2.2 建筑系统

#### 槽位机制
- 每个区域有**有限的槽位**（10-20 个）
- 建筑占用槽位（0.5-3 个不等）
- 槽位是核心资源，逼迫玩家做选择

#### 建筑类型

### 2.4 科技树系统

#### 自动化科技线
```
手工制造（起始）
 ↓
自动化 I（解锁：组装机 Mk1）
 ├→ 自动化 II（组装机 Mk2，速度 +50%）
 │   ↓
 │   自动化 III（组装机 Mk3，速度 +100%）
 │
 └→ 高级采矿（矿机效率 +50%）
     ↓
     超级采矿（矿机效率 +100%）
```

#### 军事科技线
```
基础武器（解锁：机枪无人机、普通子弹）
 ├→ 穿甲弹（对装甲 +100% 伤害）
 │   ↓
 │   爆炸物（范围伤害）
 │       ↓
 │       高爆炸药（范围 +50%，伤害 +50%）
 │
 ├→ 激光武器（需要：化工科技）
 │   ├→ 激光 II（伤害 +50%）
 │   └→ 激光 III（穿透攻击）
 │
 └→ 核武器（需要：核能科技）
     └→ 轨道打击（超高伤害单体攻击）
```

#### 工业科技线
```
基础冶炼（解锁：熔炉）
 ├→ 高级冶炼（熔炉速度 +50%）
 │   ↓
 │   钢材生产（解锁：钢板配方）
 │
 ├→ 化工科技（需要：原油，解锁：化工厂）
 │   ├→ 塑料生产
 │   ├→ 电池生产
 │   └→ 硫酸生产
 │       ↓
 │       高级化工
 │
 └→ 核能科技（需要：铀矿）
     ├→ 铀处理
     └→ 核反应堆
```

#### 基础设施科技线
```
仓储 I（仓库容量 +50%）
 ├→ 仓储 II（容量 +100%）
 │   ↓
 │   仓储 III（容量 +200%）
 │
 ├→ 紧凑建筑（某些建筑槽位 -0.5）
 │
 └→ 区域扩展（某区域 +2 槽位，可多次研究）

储能技术 I（解锁：蓄电池）
 ├→ 储能技术 II（解锁：大型蓄电池）
 └→ 超级储能（解锁：超级电容）

能效优化
 ├→ 建筑能效 I（所有建筑电耗 -20%）
 └→ 建筑能效 II（所有建筑电耗 -40%）
```

#### 指挥科技线
```
基础指挥（战斗无人机上限：10 架）
 ↓
高级指挥 I（上限：20 架）
 ↓
高级指挥 II（上限：30 架）
 ↓
超级指挥（上限：50 架）
 ↓
终极指挥（上限：100 架）
```

### 2.6 电力系统

#### 昼夜循环
```
1 个完整昼夜 = 5 分钟（实时）
白天：3 分钟
夜晚：2 分钟

太阳能发电站：
- 白天：25 电力/秒
- 夜晚：0 电力/秒

风力发电站：
- 白天：10 电力/秒
- 夜晚：15 电力/秒
```

#### 电力优先级
```
优先级 1（最高）：蓄电池充电
优先级 2（高）：矿机、研究中心
优先级 3（中）：组装机、化工厂、熔炉
优先级 4（低）：军工厂、其他建筑

电力不足时：
- 100%：全速运行
- 80%：低优先级建筑降速 20%
- 50%：低优先级建筑降速 50%
- 30%：低优先级建筑完全停机
- 0%：所有建筑停机
```

#### 电力的双重角色
```
1. 建筑运行（持续消耗）
   - 矿机：5-20 电力/秒
   - 组装机：10-25 电力/秒
   - 化工厂：20-40 电力/秒

2. 战斗消耗（爆发消耗）
   - 激光无人机：50 电力/回合
   - 等离子无人机：80 电力/回合
   - 大规模战斗可能消耗数千电力
```

#### 战斗流程

**1. 战前准备**
- 侦查下一区域（显示敌人类型和数量）
- 组建无人机编队（受指挥科技限制）
- 检查资源储备（弹药、电力）

**2. 回合制战斗**
```
回合 N：
- 我方无人机攻击
  → 消耗弹药和电力
  → 造成伤害
  → 击杀敌人

- 敌方虫族反击
  → 攻击无人机
  → 造成损失

- 更新状态
  → 检查电力是否充足
  → 检查是否胜利/失败

选项：
- [继续战斗]
- [撤退]（成功率基于剩余兵力）
```

**3. 战斗结算**
- 显示伤亡和消耗
- 占领区域
- 获得新资源点和槽位

#### 撤退机制
```
撤退成功率 = 基础 50% + 快速单位加成 - 损失惩罚

撤退成功：
- 保留剩余无人机
- 弹药消耗 50%
- 3 分钟冷却

撤退失败：
- 额外损失 20% 无人机
- 弹药全部消耗
```

### 2.8 虫族系统

## 六、核心设计理念

### 6.1 设计支柱

**1. 简化但不简陋**
- 去掉地图和物流，但保留生产链深度
- 文字界面，但信息清晰易读

**2. 策略 > 操作**
- 专注规划和优化
- 最小化重复操作

**3. 渐进式复杂度**
- 早期简单易懂
- 后期策略深度

**4. 明确的进度感**
- 每个区域都是里程碑
- BOSS 提供阶段性挑战

### 6.2 玩家体验目标

**前 30 分钟：** 理解基础循环，感受自动化的乐趣  
**1 小时：** 面对第一个 BOSS，体验准备和挑战  
**2 小时：** 建立复杂生产链，享受优化的成就感  
**3 小时：** 通关主线，解锁无尽模式

### 6.3 与 Factorio 的差异

**保留：**
✓ 生产链优化
✓ 科技树研究
✓ 资源管理
✓ 自动化建设

**简化：**
✗ 无地图系统
✗ 无物流/传送带
✗ 无空间规划
✗ 纯文字界面

**新增：**
✓ 线性区域推进
✓ 回合制战斗
✓ 槽位资源管理
✓ 昼夜电力挑战

## 十、总结

《扩张前线》是一款专注于**生产优化**和**战略推进**的策略游戏。通过简化 Factorio 的复杂系统，保留其核心乐趣，并加入线性推进和回合制战斗，创造出独特的游戏体验。

**核心乐趣：**
- 建立高效自动化生产链
- 研究科技解锁新可能
- 管理有限的槽位资源
- 征服虫族占领的区域

**目标玩家：**
- Factorio/异星工厂 爱好者
- 策略游戏玩家
- 喜欢优化和规划的玩家

**开发周期：** 10-14 周（完整版本）

**平台：** 网页游戏
