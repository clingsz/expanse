# 🎨 扩张前线 - 多彩主题系统

## 配色方案

每个Tab都有独特的霓虹色主题：

### 🔵 区域 (Region) - 霓虹青
- **主色**: `#00f3ff` (Cyan)
- **效果**: 默认主题，科技感十足的青色光芒
- **用途**: 区域管理、建筑查看

### 🟠 建造 (Build) - 霓虹橙
- **主色**: `#ff9500` (Orange)
- **效果**: 温暖的橙色，代表建设与创造
- **用途**: 建筑建造界面

### 🟣 科技 (Tech) - 霓虹紫
- **主色**: `#b967ff` (Purple)
- **效果**: 神秘的紫色，象征科技树与研究
- **用途**: 科技研发界面

### 🟢 仓库 (Storage) - 霓虹绿
- **主色**: `#00ff88` (Green)
- **效果**: 明亮的绿色，代表资源与库存
- **用途**: 仓库管理界面

### 🟡 地图 (Map) - 霓虹黄
- **主色**: `#ffdd00` (Yellow)
- **效果**: 醒目的黄色，象征探索与导航
- **用途**: 地图与区域探索

## 实现细节

### 动态主题切换
- 切换Tab时，整个界面的主题色会平滑过渡（0.5秒）
- 影响的元素：
  - 页面标题颜色
  - 面板边框和发光效果
  - 建筑卡片边框
  - 激活的Tab按钮背景色
  - 所有主要UI元素

### Tab栏颜色指示
- 每个Tab按钮顶部有3px的颜色边框，显示其主题色
- 鼠标悬停时，按钮文字和背景会变为该Tab的主题色
- 激活状态下，按钮以主题色填充

### 视觉效果
- 所有颜色都带有霓虹发光效果
- 平滑的颜色过渡动画
- 保持赛博朋克美学风格

## 使用方法

主题通过 `data-theme` 属性自动切换：
```javascript
// 切换到建造主题
document.body.setAttribute('data-theme', 'build');
```

CSS变量会自动更新：
```css
body[data-theme="build"] {
    --primary-color: #ff9500;
    --border-glow: 0 0 8px rgba(255, 149, 0, 0.4);
}
```

所有使用 `var(--primary-color)` 的元素会自动应用新颜色！
