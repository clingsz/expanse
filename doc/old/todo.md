## 目标
搭建基础框架，实现一个可运行的界面雏形，包含核心循环的最小可行版本。

## Day 1 最小可行版本目标
实现一个可以运行的游戏原型，包含：
1. ✓ 区域界面显示基础信息
2. ✓ 资源系统（铁、铜、煤、电力）
3. ✓ 建造系统（可建造矿机和熔炉）
4. ✓ 生产循环（矿机自动产出资源）
5. ✓ 界面实时更新

## 开发优先级
1. **高优先级**: 区域界面 + 资源系统 + 建造系统
2. **中优先级**: 建筑管理 + 生产配方
3. **低优先级**: 科技树、战斗系统（Day 2+）

## 技术债务和后续改进
- 配方系统（熔炉：矿石→金属板）
- 电力系统完善
- 昼夜循环
- 科技树
- 战斗系统
